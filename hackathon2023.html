Nuix Hackathon 2023
"Flexible Query Builder Demo" by Anthony
  • The Flexible Query Builder enhances Nuix Investigate by introducing a new feature that enables users to construct search queries directly from an existing file, eliminating the need for manual typing input.
  • During the demo, I uploaded a live photo of my manager and myself, and the app automatically extracted metadata to search for files containing geographical data within a 10-mile radius along with other parameters.
  • This feature significantly streamlines the search process, making it faster and more user-friendly, particularly for complex queries involving geospatial information."
